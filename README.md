# Sacred Light Symphony

A refined cultural initiative inspired by gratitude, presence, connection, and
artistic expression — rooted in Küsnacht, Switzerland and the legacy of Tina Turner.

> Reveal the feeling, not the mechanism. If in doubt, simplify.

## Stack

React 19 · TypeScript · Vite 6 · react-router-dom 7 · CSS Modules · Supabase
(planned) · Netlify hosting.

## Develop

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # type-check + production build to dist/
npm run preview  # preview the production build
```

## Deploy

Deployed on Netlify via continuous deployment. Build command `npm run build`,
publish directory `dist`; SPA redirects configured in `netlify.toml`.
